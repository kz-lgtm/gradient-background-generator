# gradient-background-generator

## Overview
A web tool that allows users to generate and preview CSS gradient backgrounds by selecting two colors, with the ability to copy the generated CSS code.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Interactive color selection using color pickers
- Real-time gradient preview
- Displays generated CSS gradient code
- Copy-to-clipboard functionality

## What I Learned
- Handling user input and events in JavaScript
- Dynamically updating styles based on user interaction
- Working with the clipboard API
- Creating responsive and interactive UI elements

## Notes
This project was inspired by a tutorial and extended by adding a copy-to-clipboard feature for improved usability.
